다음은 Python 2와 Python 3에서 클래스를 정의하는 방법입니다.

```python
# Python 2
class MyClass(object):
    def __init__(self, data):
        self.data = data
```

```python
# Python 3
class MyClass:
    def __init__(self, data):
        self.data = data
```
